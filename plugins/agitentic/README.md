# agitentic plugin

The agitentic plugin. Ships agentic git/GitHub helpers as
[agentskills.io](https://agentskills.io)-compliant skills.

Skills:

- **`agitentic:git-fork`** — fork a GitHub repo and clone it with the
  contributor-style two-remote layout (`upstream` → original repo,
  `fork` → your fork). The local default branch tracks `upstream`.

For installation and usage, see the main repo README:
<https://github.com/robobryce/agitentic>
